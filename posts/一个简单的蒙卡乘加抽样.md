乘加抽样问题想明白了其实就很简单，但想不明白很容易绕进去。

分布密度函数其实离散化就是概率，也就是画一个频率的直方图即可作为子样分布图，画一个密度函数曲线即可作为理论分布图：

```python
import matplotlib.pyplot as plt

def plot_histogram(samples, pdf_func, title, x_range, bins=50):
    """辅助绘图函数"""
    plt.figure(figsize=(8, 4))
    # 绘制抽样结果直方图
    plt.hist(samples, bins=bins, density=True, alpha=0.6, color='g', label='Simulation')
    # 绘制曲线
    x = np.linspace(x_range[0], x_range[1], 200)
    y = pdf_func(x)
    plt.plot(x, y, 'r-', lw=2, label='Theoretical distribution')
    plt.title(title)
    plt.xlabel('x')
    plt.ylabel('Density')
    plt.legend()
    plt.grid(True, alpha=0.3)
    plt.show()
```

一个乘方分布$f(x)=f_1(x)H_1(x)+f_2(x)H_2(x)$的抽样：

<img src="https://gitee.com/wenswuu/pictures/raw/master/20251119000832903.webp" style="zoom:50%;" />

它的流程应该是，先抽出一个随机数$\xi_1$，然后进行第一层判断，如果是$\le$，则要抽出$X_{f_1}$，此时应该要再次抽出一个随机数$\xi_2$得到$X_{f_1}$，再抽一个随机数$\xi_3$，进行第二层判断，如果是$\le$，则得到一个$X_f$。

第一层判断其实不存在"舍去"，因为有两条线，$\xi_1$其实决定了抽样的总次数，但到了第二层是否取值则存在一定的概率，$\xi_2,\xi_3$决定了最终子样的个数，故也可能这几个随机数都废了，那就进行下一次抽样，最终得到满足$f$分布的子样。

明白了这个流程再用代码实现就变得很清晰了，$\alpha$是散射前的能量，相当于一个已知量，$\xi_2$抽取的结果决定了从$f_1(x),f_2(x)$中抽取$X_{f_1},X_{f_2}$的结果，虽然都用$\xi_2$表示，但其实数值不相等，每次都要重新抽取。

```py
# 第三题
import numpy as np
alpha=2 #alpha值可以改为任意的能量
samples=1000000
xi1=np.random.rand(samples) #生成随机数
X_f=[]

#定义K(alpha)函数
def func_K(alpha):
    y=(2*(alpha+1)/(alpha**2))
    y=(1-y) * np.log(1+2*alpha)+0.5+(4/alpha)-(1/(2*((2*alpha+1)**2)))
    return y

#定义本题的分布密度函数
def func(x,alpha):
    y=(((alpha+1-x)/alpha)**2)
    y=y+(1/x)
    y=y/(x**2)
    y=y+(1/x)-(1/(x**2))
    y=y/(func_K(alpha))
    return y

# 定义H1/M1函数
def func1(x, alpha):
    y=((alpha+1-x)/alpha)**2
    y=y+(1/x)
    y=(1/2)*y
    return y

# 定义H2/M2函数
def func2(x):
    y=(1/x)-(1/(x*2))
    y=4*y
    return y

for xi1 in xi1:
    if xi1 <= (8/(9+2*alpha)):
        xi2= np.random.rand(1) #生成一个随机数xi2
        x1=((1+2*alpha)/(1+2*alpha*xi2)) #抽样得到X_{f_1}
        xi3= np.random.rand(1)
        if xi3 <= func1(x1, alpha):
            X_f.append(x1)
    else:
        xi2= np.random.rand(1) #生成一个随机数xi2
        x2=(2 * alpha * xi2 +1) #抽样得到X_{f_2}
        xi3= np.random.rand(1)
        if xi3 <= func2(x2):
            X_f.append(x2)

X_f = np.array(X_f[:])
            
plot_histogram(X_f, lambda x: func(x, alpha), 
               "Sampling", [1, 1+2*alpha])
```
