# MyBlog

A reading-first static blog inspired by the structure and restraint of Lilian Weng's website.

## Local preview

Open `index.html` directly in a browser.

## Publish to GitHub Pages

1. Create a GitHub repository and push this folder to the `main` branch.
2. In GitHub, open `Settings -> Pages`.
3. Set the source to `GitHub Actions`.
4. Push new commits to `main` to trigger deployment.

## Add a new post

1. Copy one of the files in `posts/`.
2. Update the title, metadata, and article body.
3. Add the post link to `index.html`, `archive.html`, and optionally `tags.html`.
4. If you want search coverage, add the entry to `assets/search.js`.
