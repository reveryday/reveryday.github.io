CGT48自动购票脚本-initial

（需手动登入 仅限购一张）

运行前需配置：

  1. python环境：安装anaconda jupyter notebook

  2. Selenium环境：chrome打开jupyter notebook出现了一些问题，所以用了edge，需安装好edge对应版本的edgedriver，用指令pip install selenium安装好selenium库

用edge打开jupyter notebook，新建一个.ipynb文件，运行以下代码即可

#CII首演第一场抢票脚本
import time
from bs4 import BeautifulSoup
from selenium import webdriver
from selenium.webdriver.edge.service import Service
from selenium.webdriver.common.by import By

driver_path = 'D:\\app data\\edgedriver\\edgedriver_win64\\msedgedriver.exe'
service = Service(executable_path=driver_path)
browser = webdriver.Edge(service=service)

target_url="https://48.gnz48.com/tickets/item/6774?seat_type=3"  
#进入抢票页面
browser.get(target_url)
html_k = browser.execute_script('return document.documentElement.outerHTML')  # 获取当前页面的HTML
bs_k = BeautifulSoup(html_k, 'html.parser') 

# seattype2：VIP座票；seattype3：普通座票；seattype4：普通站票
# 选择座位类型
theseattype = browser.find_element(By.XPATH, '//em[@id="seattype3"]')
browser.execute_script('arguments[0].click();', theseattype)

#定时购买
click_time = 0
while True:  # 无限循环
    if click_time > 0:
        break
    if time.strftime("%Y-%m-%d %H:%M:%S",time.localtime()) >= '2025-02-05 20:30:00':  # 设置抢票时间
        try:
            WebDriverWait(browser, 5).until(
            EC.text_to_be_present_in_element((By.ID, 'buy'), '立即购买')
            )
            determine = WebDriverWait(browser, 5).until(
            EC.element_to_be_clickable((By.ID, 'buy'))
            )
            browser.execute_script('arguments[0].click();', determine)
            click_time += 1
        except Exception as e:
            print(f"未找到购买按钮，继续尝试: {e}")

首演切票试了一下，毫无作用。

问题原因：定时购买前的进入网页手动登入等都没有问题，在20：30前约15s左右运行最后一段代码时，网页端一直没有反应，输出端也无任何输出。
