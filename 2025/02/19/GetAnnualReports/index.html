一个简单的自动筛选文件脚本

帮朋友写了一个很简单的自动筛选文件的python脚本：在stock_code.csv文件中有需要的公司对应的股票代码，而annual_report文件夹中则是所有上市公司的年报（皆为txt文件），文件名前六位即为公司股票代码，所以需要根据csv文件中的股票代码筛选出对应公司的年报，并复制到一个新的文件夹里：

import os
import pandas as pd
import shutil

# 读取股票代码，设置dtype为str确保保留前导零
stock_codes = pd.read_csv('stock_code.csv', header=None, dtype=str)[0].tolist()

# 年报文件所在的目录
annual_report_dir = 'annual_report'
# 新建目标文件夹
target_dir = 'filtered_reports'

# 如果目标文件夹不存在，创建它
if not os.path.exists(target_dir):
    os.makedirs(target_dir)

# 筛选并复制匹配的年报文件
matching_files = []

for filename in os.listdir(annual_report_dir):
    # 获取文件名前六位
    stock_code = str(filename[:6])
    if stock_code in stock_codes:
        matching_files.append(filename)
        # 构建源文件和目标文件的完整路径
        source_file = os.path.join(annual_report_dir, filename)
        target_file = os.path.join(target_dir, filename)
        # 复制文件
        shutil.copy2(source_file, target_file)

# 输出匹配的文件
print(f"已将以下匹配的年报文件复制到 {target_dir} 文件夹:")
for file in matching_files:
    print(file)
print(f"\n共复制了 {len(matching_files)} 个文件")

这段代码虽然简单，但用初版运行的时候也存在一些问题，debug还是得要耐心呀，需一部分一部分慢慢测试，发现最初无法成功的原因在于，csv文件中提取得来的stock_codes是str类型，跟filename提取前六位（filename[:6]）得到的数据类型不一样，所以最简单的强制转换一下数据类型就成功了。
