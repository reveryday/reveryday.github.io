Linear Regression-scratch

import random
import torch
from d2l import torch as d2l
  1. 生成数据集 - 根据带有噪声的线性模型构造一个人造数据集
def synthetic_data(w, b, num_examples): 
    """生成y=Xw+b+噪声"""
    X = torch.normal(0, 1, (num_examples, len(w)))
    y = torch.matmul(X, w) + b
    y += torch.normal(0, 0.01, y.shape)
    return X, y.reshape((-1, 1))
    
true_w = torch.tensor([2, -3.4])
true_b = 4.2
features, labels = synthetic_data(true_w, true_b, 1000)

其中：

torch.normal(mean, std, size)：生成 N(mean, std^2) （正态）分布的随机数

  1. 读取数据集 - 每次抽取一小批量样本，对数据集进行遍历
def data_iter(batch_size, features, labels):
    num_examples = len(features)
    indices = list(range(num_examples))
    # 这些样本是随机读取的，没有特定的顺序
    random.shuffle(indices)
    for i in range(0, num_examples, batch_size):
        batch_indices = torch.tensor(
            indices[i: min(i + batch_size, num_examples)])
        yield features[batch_indices], labels[batch_indices]

其中：

random.shuffle：用于打乱列表中的元素

  1. 初始化模型参数
w = torch.normal(0, 0.01, size=(2,1), requires_grad=True)
b = torch.zeros(1, requires_grad=True)

其中：

torch.zeros：用于创建全0张量的函数

  1. 定义模型 - 线性模型
def linreg(X, w, b):  #@save
    """线性回归模型"""
    return torch.matmul(X, w) + b
  1. 定义损失函数（均方损失）- 需计算损失函数的梯度
def squared_loss(y_hat, y):  #@save
    """均方损失"""
    return (y_hat - y.reshape(y_hat.shape)) ** 2 / 2
  1. 定义优化算法
def sgd(params, lr, batch_size):
    """小批量随机梯度下降"""
    with torch.no_grad():
        for param in params:
            param -= lr * param.grad / batch_size
            param.grad.zero_()
  1. 训练
lr = 0.03
num_epochs = 3
net = linreg
loss = squared_loss

for epoch in range(num_epochs):
    for X, y in data_iter(batch_size, features, labels):
        l = loss(net(X, w, b), y)  # X和y的小批量损失
        # 因为l形状是(batch_size,1)，而不是一个标量。l中的所有元素被加到一起，
        # 并以此计算关于[w,b]的梯度
        l.sum().backward()
        sgd([w, b], lr, batch_size)  # 使用参数的梯度更新参数
    with torch.no_grad():
        train_l = loss(net(features, w, b), labels)
        print(f'epoch {epoch + 1}, loss {float(train_l.mean()):f}')

其中： （超参数设置很棘手，需要通过反复试验进行调整)

Linear Regression-concise

  1. 生成数据集
import numpy as np
import torch
from torch.utils import data
from d2l import torch as d2l

true_w = torch.tensor([2, -3.4])
true_b = 4.2
features, labels = d2l.synthetic_data(true_w, true_b, 1000)
  1. 读取数据集 - 调用框架中现有的API来读取数据，将features和labels作为API的参数传递
def load_array(data_arrays, batch_size, is_train=True): 
    """构造一个PyTorch数据迭代器"""
    dataset = data.TensorDataset(*data_arrays)
    return data.DataLoader(dataset, batch_size, shuffle=is_train)
    
batch_size = 10
data_iter = load_array((features, labels), batch_size)

其中：

data.TensorDataset：用于将多个张量（Tensor）打包成一个数据集的工具类，提供了一种简单的方式将输入数据（例如特征和标签）组合成一个可迭代的数据集对象

data.DataLoader：数据迭代器，布尔值 is_train 表示是否希望数据迭代器对象在每个迭代周期内打乱数据

  1. 定义模型
# nn是神经网络的缩写
from torch import nn

net = nn.Sequential(nn.Linear(2, 1))

nn.Sequential(nn.Linear(2, 1)) ：是、一个非常简单的模型，它表示一个单层全连接神经网络（也叫线性层），输入是2 个特征，输出是1 个特征。即有一个权重矩阵和偏置向量，它会对输入的 2 维数据进行线性变换，得到一个 1 维的输出。

  1. 初始化模型参数
net[0].weight.data.normal_(0, 0.01)
net[0].bias.data.fill_(0)

net 是一个 nn.Sequential 模型，net[0] 是该模型中的第一个层，net[0].weight 是该层的权重矩阵，weight.data 是访问权重张量的原始数据部分，normal_() 是一个原地操作（用下划线结尾表示原地操作），它会用均值为0，标准差为 0.01 的正态分布来初始化权重

  1. 定义损失函数
loss = nn.MSELoss()

计算均方误差使用的是MSELoss类，也称为平方L2范数，默认情况下，它返回所有样本损失的平均值

  1. 定义优化算法
trainer = torch.optim.SGD(net.parameters(), lr=0.03)

在 PyTorch 中创建一个优化器，使用 随机梯度下降（SGD）算法来优化神经网络 net 的参数，其中，torch.optim.SGD：这是 PyTorch 中实现的随机梯度下降（SGD）优化器，net.parameters()：这是一个返回模型所有参数（权重和偏置）的迭代器

  1. 训练
num_epochs = 3
for epoch in range(num_epochs):
    for X, y in data_iter:
        l = loss(net(X) ,y)
        trainer.zero_grad()
        l.backward()
        trainer.step()
    l = loss(net(features), labels)
    print(f'epoch {epoch + 1}, loss {l:f}')

通过深度学习框架的高级API来实现模型只需相对较少的代码，不必单独分配参数、不必定义损失函数，也不必手动实现小批量随机梯度下降，当需要更复杂的模型时，高级API的优势将大大增加！
