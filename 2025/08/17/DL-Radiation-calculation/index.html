DL & Radiation calculation

发表于2025-08-17|更新于2025-09-06
  1. 目标：基于机器学习（Transformer）实现对辐射屏蔽参数的快速计算。
  2. 解决了什么问题？
    • 核设施退役：（1）作业方案剂量评估和优化的基础。（2）在发生核事故时，对核电站状态的快速且准确评估。
    • 源项反演：在核事故期间，如何快速且准确地估算放射性物质的释放源项，已成为核应急响应早期阶段的一项紧迫的科学问题。
  3. 方案：Transformer+BUF
  4. 创新点：
    • Transformer模型
    • 能计算大尺度问题

Gamma辐射场理论计算

  1. Buildup Factor

    $$
    I=B\cdot I_0 e^{-\mu d}
    $$

    累积因子作为gamma辐射场一个关键辐射量，将其作为切入点与模型的输出是非常合理的。计算需要解决的问题：

    • 大尺度预测精度不足
    • 几何因素的影响（屏蔽结构、交界面处的影响）
    • 反射效应的影响（源附近散射与探测器外侧介质散射）

  2. MCNP

（精确的累积因子测量成本高且数据量有限，而机器学习对大规模训练有着很高的数据需求）

DL-BUF

需要调研：

  1. ML得到GP拟合公式【单层材料】【累积因子】的新形式，主要是新公式，GP公式在算参数时出现了很多异常现象。这种形式自从上世纪八十年代推出后很久都没更新，一定存在新的形式，先看看有没有新的公式。
  2. 机器学习算单层累积因子，聚焦于材料伽马射线屏蔽性质的文章，预测新材料的伽马屏蔽能力（就是累积因子），且评估工作量，数据集：28种材料，每种材料26个能点，每个能点28个数据，六种累积因子（聚焦一个就行）

GP formula-BUF

落地形态

Road-map

Data-generation

  • high-quality data representing radiation fields under various conditions,

  • gather experimental measurements for validation

Data-preprocession

  • Normalize and clean the data.

  • Consider techniques like augmentation if the dataset is small.

Model selection and development

  • For spatial data or field distributions, convolutional neural networks (CNNs) might be effective. If involves sequential or multi-dimensional data, architectures like transformers or recurrent networks could be considered.

  • Begin with simpler models to establish a baseline. Gradually increase model complexity to capture the nuances in the data.

Integration into Software

  • Develop visualization tools to display radiation field distributions.
  • Write comprehensive documentation for future developers and users.
  • Package your software for the intended environment
