安装VS2022与oneAPI

VS2022安装时只需勾选：

oneAPI安装：

访问官网（网址：https://www.intel.cn/content/www/cn/zh/developer/tools/oneapi/toolkits.html），先安装基础工具包 Intel oneAPI Base Toolkit，再安装 Intel oneAPI HPC Toolkit

安装Intel oneAPI Base Toolkit时，如果全部安装所占内存太大了，电脑要爆了，所以我只勾选了两个包：

安装Intel oneAPI HPC Toolkit 我只勾选了一个包：

![](D:\Program Files\HexoBlog\blog\source_posts\images\vs03.png)

在VS2022上测试时发现无法新建一个Fortran项目（没有Fortran语言选项），可能是Intel oneAPI HPC Toolkit没有勾选Intel Fortran Compiler（？）

VS安装这两个插件就解决了：

![](D:\Program Files\HexoBlog\blog\source_posts\images\vs04.png)

建立Fortran项目测试：

新建一个Fortran语言的Main Program Code项目，会出现内置的Fortran示例代码 （.f90文件），运行测试即可。

贴一下可用于测试的示例代码：

program Console1 implicit none print *, 'Hello World' pause end program Console1
