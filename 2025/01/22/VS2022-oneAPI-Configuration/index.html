oneAPI安装：

在VS2022上测试时发现无法新建一个 Fortran 项目（没有 Fortran 语言选项），可能是Intel oneAPI HPC Toolkit安装时没有勾选Intel Fortran Compiler（？）

建立Fortran项目测试：

新建一个 Fortran 语言的 Main Program Code 项目，会出现内置的 Fortran 示例代码 （.f90文件），运行测试即可。

贴一下可用于测试的示例代码：

    implicit none 
    print *, 'Hello World'
    pause 
end program Console1
