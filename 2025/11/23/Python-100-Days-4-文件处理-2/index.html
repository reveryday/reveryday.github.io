1-Python读写CSV

CSV（Comma Separated Values）全称逗号分隔值文件是一种简单、通用的文件格式，被广泛的应用于应用程序（数据库、电子表格等）数据的导入和导出以及异构系统之间的数据交换。因为CSV是纯文本文件，不管是什么操作系统和编程语言都是可以处理纯文本的，而且很多编程语言中都提供了对读写CSV文件的支持，因此CSV格式在数据处理和数据科学中被广泛应用。

将数据写入CSV

可以使用Python标准库中的csv模块，该模块的writer函数会返回一个csvwriter对象，通过该对象的writerow或writerows方法就可以将数据写入到CSV文件中；

import csv

with open("test.csv","w") as file:
    writer = csv.writer(file)
    writer.writerow(["ML","DL","RL"])

从CSV读取数据

如果要读取刚才创建的CSV文件，可以使用下面的代码，通过csv模块的reader函数可以创建出csvreader对象，该对象是一个迭代器，可以通过next函数或for-in循环读取到文件中的数据。

with open("test.csv","r") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

如需使用Python做数据分析，很有可能会用到名为pandas的三方库，它是Python数据分析的神器之一。pandas中封装了名为read_csv和to_csv的函数用来读写CSV文件，其中read_CSV会将读取到的数据变成一个DataFrame对象，而DataFrame就是pandas库中最重要的类型，它封装了一系列用于数据处理的方法（清洗、转换、聚合等）；而to_csv会将DataFrame对象中的数据写入CSV文件，完成数据的持久化。read_csv函数和to_csv函数远远比原生的csvreader和csvwriter强大。

import pandas as pd
data = pd.read_csv("test.csv",header=None)
print(data)
#    0   1   2
#0  ML  DL  RL

2-Python读写Excel文件

可以使用openpyxl库，这个库不仅仅可以操作Excel，还可以操作其他基于 Office Open XML 的电子表格文件。

读Excel文件

import openpyxl
import datetime

wb=openpyxl.load_workbook('src/test.xlsx') #加载一个工作簿
sheet = wb.worksheets[0] #获取工作表
print(wb.sheetnames)
print(sheet.max_row, sheet.max_column)
print(sheet.cell(1, 2).value)  #获取指定单元格的值

读取所有数据很繁琐的感觉，存数据什么的还是不用Excel比较好。

写Excel文件

# write excle file
import openpyxl
wb = openpyxl.Workbook() # create a new workbook
sheet = wb.active  #添加工作表
sheet.title = "Course"  #修改工作表名称
titles=["高反","数分","核物理","MC"]
for i in range(4):
    sheet.cell(1,i+1).value = titles[i]
    sheet.cell(2,i+1).value = "1"

wb.save("src/course.xlsx")  #保存文件

3-操作Word和PPT

操作word，安装第三方库：

pip install python-docx
from docx import Document
from docx.shared import Cm, Pt

from docx.document import Document as Doc

document = Document() #创建一个Document对象
document.add_heading('很烦', 0) #添加标题
p=document.add_paragraph('这是第一段，') #添加段落
p.add_run('为什么间歇性茫然') #添加一句话

document.save('test.docx') #保存文件

对于一个已经存在的 Word 文件，可以通过下面的代码去遍历它所有的段落并获取对应的内容：

from docx import Document
from docx.document import Document as Doc

doc = Document('test.docx')  # type: Doc
for no, p in enumerate(doc.paragraphs):
    print(no, p.text)

操作PowerPoint，需安装名为python-pptx的三方库，生成powerpoint：

from pptx import Presentation

pres=Presentation() #创建PPT对象
title_slide_layout = pres.slide_layouts[0]
slide = pres.slides.add_slide(title_slide_layout) #添加幻灯片

title = slide.shapes.title #获取标题
title.text = '这是一页PPT' #设置标题文字
subtitle = slide.placeholders[1] #获取副标题
subtitle.text = "用python生成的" #设置副标题文字

pres.save('test.pptx') #保存PPT文件
