Day01-03 - 2025/11/1

变量和类型

变量是数据的载体，简单的说就是一块用来保存数据的内存空间，变量的值可以被读取和修改。变量名由字母、数字和下划线构成，数字不能开头。

数据类型 int float str bool

Python 中可以处理任意大小的整数，而且支持二进制、八进制、十进制和十六进制的表示法。

Day04 - 2025/11/2

运算符

运算符 描述
[]、[:] 索引、切片
~ 按位取反（-x-1）
**、*、/、%、// 幂、乘、除、模、整除
>>、<< 右移、左移
is、is not 身份运算符
in、not in 成员运算符
not、or、and 逻辑运算符
**=、&=、^=、>>=、<<= 赋值运算符

其中，切片[:]的取值是左闭右开：

b=[0,1,2,3,4,5,6]
print(b[2:5])
# 输出[2, 3, 4]

例：在处理Dataset时

data=pd.read_csv(file_path, header=None)
X=data.iloc[:7]       #取前7行
Y=data.iloc[:, :7]   #取前7列
写法 等价于 含义
df.iloc[:7] df.iloc[:7, :] 前 7 行，所有列
df.iloc[:, :7] 所有行，前 7 列
df.iloc[:7, :3] 前 7 行 + 前 3 列

iloc 是 纯位置索引，语法：df.iloc[行位置, 列位置]，只写一部分时，缺失的维度默认全选。
