Python-100-Days-1-Basics

发表于2025-11-01|更新于2025-11-30|Python-100-Days

教材：https://github.com/jackfrued/Python-100-Days/tree/master

1_变量和类型

变量是数据的载体，简单的说就是一块用来保存数据的内存空间，变量的值可以被读取和修改。变量名由字母、数字和下划线构成，数字不能开头。

数据类型 int float str bool

Python 中可以处理任意大小的整数，而且支持二进制、八进制、十进制和十六进制的表示法。

2_运算符

运算符 描述
[] 、 [:] 索引、切片
~ 按位取反（-x-1）
** 、 * 、 / 、 % 、 // 幂、乘、除、模、整除
>> 、 << 右移、左移
is 、 is not 身份运算符
in 、 not in 成员运算符
not 、 or 、 and 逻辑运算符
**= 、 &= 、 ^= 、 >>= 、 <<= 赋值运算符

其中，切片 [:] 的取值是左闭右开；

b=[0,1,2,3,4,5,6]
print(b[2:5])
# 输出[2, 3, 4]

例：我在处理Dataset时，iloc是 纯位置索引，语法：df.iloc[行位置, 列位置]，只写一部分时，缺失的维度默认全选。

data=pd.read_csv(file_path, header=None)
X=data.iloc[:7]       #取前7行
Y=data.iloc[:, :7]   #取前7列
写法 等价于 含义
df.iloc[:7] df.iloc[:7, :] 前 7 行，所有列
df.iloc[:, :7] 所有行，前 7 列
df.iloc[:7, :3] 前 7 行 + 前 3 列

3_分支结构

  1. if-else
  2. match-case
if:
elif:
else:

       print("太轻了")

4_循环结构

1-for in循环：

import time
for _ in range(22):
    print("It is time to go to sleep")
    time.sleep(1)

2-while循环：

i=0
while i<10:
    i +=1
    print(i)

Python 没有 ++ （自增）或 – （自减）这种操作符，我竟然才知道，真的很少自己手搓代码。

3-break和continue:

i=0
total=0
while i<10:
    i +=1
    if i%2==0:
        continue
    total += i
print(total)

5_常用数据结构-列表（list）

列表：由一系元素按特定顺序构成的数据序列。

item = [1, 2, 3, 4, 5]   # 列表（list）
item1 = {1, 2, 3, 4, 5}  # 集合（set）
item2=(1,2,3,4,5) # tuple
item3="1,2,3,4,5" #str
list （列表） tuple（元组） set （集合） Str（字符串）
有序 有序 无序 有序
允许重复元素 不允许重复元素 允许
支持索引访问： item[0] 支持索引访问 不支持索引访问 支持索引
可变 适合多线程环境、不可变类型在创建时间上优于对应的可变类型 可变
计算符（-a） 操作
+/* 两个列表的拼接/列表的重复运算
in/not in 判断一个元素在不在列表中
a[0] 索引
a[start: end: stride] 切片
a.append()/a.insert() 插入元素
a.remove()/a.pop()/a.clear() 删除指定元素/清空列表
a.index(“element”,1) （从索引位置1开始）查找element在列表中的索引位置
a.count(“element”) 统计element出现次数
a.sort()/a.reverse() 排序/反转

使用 + 运算符实现两个列表的拼接，使用 * 运算符实现列表的重复运算，使用 in 或 not in 运算符判断一个元素在不在列表中；

print(11 in item1)
# False

正向索引， [0] 可以访问列表中的第一个元素， [N - 1] 可以访问最后一个元素；对于反向索引， [-1] 可以访问列表中的最后一个元素， [-N] 可以访问第一个元素，越界的情况会出现IndexError错误；

一次性访问列表中的多个元素，可使用切片运算：形如 [start:end:stride] 的运算符， end 代表访问列表元素的终止位置（终止位置的元素无法访问）：

print(item1[1:3:1])
#[2, 3]

其中，如果 start 值等于 0 时可以将其省略；如果 end 值等于 N 时可以将其省略；如果 stride 值等于 1 时也可以将其省略。

元素的遍历有两种方法：

使用列表的 append 方法向列表中追加元素，将元素添加到列表的末尾，使用 insert 方法向列表中插入元素，在指定的位置添加新元素；

item1.append(6)
item1.insert(0,0)
print(item1)
#[0, 1, 2, 3, 4, 5, 6]

用列表的 remove 或者 pop （ pop 方法默认删除列表中的最后一个元素）从列表中删除指定元素， clear 方法，可以清空列表中的元素；

s=["a","b","c","a"]
s.remove("a")  #删的是第一个
print(s)
#['b', 'c', 'a']

列表的 index 方法可以查找某个元素在列表中的索引位置，如果找不到指定的元素， index 方法会引发 ValueError 错误；列表的 count 方法可以统计一个元素在列表中出现的次数；

s=["a","b","c","a"]
print(s.index("a",1))
print(s.count("a"))
#3
#2

列表的 sort 操作可以实现列表元素的排序，而 reverse 操作可以实现元素的反转；

s.sort()
print(s)
s.reverse()
print(s)
#['a', 'a', 'b', 'c']
#['c', 'b', 'a', 'a']

列表生成式：

嵌套列表：

a=[1,2,3]
b=[11,22,33]
c=[111,222,333]
d=[a,b,c]
print(d)
print(d[0][0])
#[[1, 2, 3], [11, 22, 33], [111, 222, 333]]
#1

6_常用数据结构-元组（tuple）

元组也是多个元素按照一定顺序构成的序列。元组和列表的不同之处在于，元组是不可变类型，其类型的变量一旦定义，其中的元素不能再添加或删除，而且元素的值也不能修改，元组与列表是可以互相转换的。

如果元组中只有一个元素，需要加上一个逗号：

m=(1,)
n=(1)
print(type(m))
print(type(n))
#<class 'tuple'>
#<class 'int'>

把多个用逗号分隔的值赋给一个变量时，多个值会打包成一个元组类型，把一个元组赋值给多个变量时，元组会解包成多个值然后分别赋给对应的变量；

在解包时，如果解包出来的元素个数和变量个数不对应，会引发 ValueError 异常，错误信息为： too many values to unpack （解包的值太多）或 not enough values to unpack （解包的值不足）。通过星号表达式，可以让一个变量接收多个值，用星号表达式修饰的变量会变成一个列表，列表中有0个或多个元素；其次，在解包语法中，星号表达式只能出现一次。

m1=(1,2,3,4,5,6)
i,*j,k=m1
print(i,j,k)
#1 [2, 3, 4, 5] 6

7_常用数据结构-字符串（str）

字符串，就是由零个或多个字符组成的有限序列。在字符串中使用 \ （反斜杠）来表示转义，Python 中有一种以 r 或 R 开头的字符串，这种字符串被称为原始字符串；

print("hello \nworld")
print(r"hello \nworld")
#print("hello \nworld")
print(r"hello \nworld")

字符串的特殊表示：Python 中还允许在 \ 后面还可以跟一个八进制或者十六进制数来表示字符，在 \u 后面跟 Unicode 字符编码；

字符串的运算：可以使用 + 运算符来实现字符串的拼接，可以使用 * 运算符来重复一个字符串的内容，可以使用 in 和 not in 来判断一个字符串是否包含另外一个字符串，用 [] 和 [:] 运算符从字符串取出某个字符或某些字符。

字符串的比较：两个字符串类型的变量，可以用比较运算符来比较两个str的大小，字符串在计算机内存中也是以二进制形式存在的，所以字符串的大小比较比的是每个字符对应的编码的大小。

字符串方法：字符串方法 .capitalize() 不会原地修改字符串，而是返回一个新的字符串。

a.title() a.capitalize() a.upper（） a.lower()
字符串每个单词首字母大写 字符串首字母大写 字符串变大写 字符串变小写

8_常用数据结构之集合（set）

集合需满足：（1）无序性，（2）互异性，（3）确定性；

集合中的元素必须是 hashable 类型，所谓 hashable 类型指的是能够计算出哈希码的数据类型，通常不可变类型都是 hashable 类型，如整数（ int ）、浮点小数（ float ）、布尔值（ bool ）、字符串（ str ）、元组（ tuple ）等。可变类型都不是 hashable 类型，因为可变类型无法计算出确定的哈希码，所以它们不能放到集合中。

元素的遍历：

# 集合
set1={5,2,3,4,5}
for i in set1:
    print(i)
#2
#3
#4
#5

集合的运算：可以求交集，并集差集等等，可以用 == 和 != 进行相等性判断，Python 为集合类型提供了判断子集和超集的运算符，就是熟悉的 < 、 <= 、 > 、 >= 这些运算符。

不可变集合：Python 中还有一种不可变类型的集合，名字叫 frozenset ，除了不能添加和删除元素， frozenset 在其他方面跟 set 是一样的。

9_常用数据结构之字典（dict）

Python 中创建字典使用 {} 字面量语法，跟集合是一样的，但字典的 {} 中的元素是以键值对的形式存在的，每个元素由 : 分隔的两个值构成， : 前面是键， : 后面是值；

# 字典
person = {
    'name': '王大锤',
    'age': 55,
    'height': 168,
    'weight': 60,
    'addr': '成都市武侯区科华北路62号1栋101', 
    'tel': '13122334455',
    'emergence contact': '13800998877'
}
print(person)
person1 = dict(name='王大锤', age=55, height=168, weight=60, addr='成都市武侯区科华北路62号1栋101')
print(person1)

可使用 for 循环对字典的键进行遍历，对于字典类型来说，成员运算和索引运算是很重要的，字典中因为保存的是键值对，所以字典需要用键去索引对应的值。字典中的键必须是不可变类型，例如整数（ int ）、浮点数（ float ）、字符串（ str ）、元组（ tuple ）等类型，这一点跟集合类型对元素的要求是一样的。

字典的方法： get 方法可以通过键来获取对应的值，使用 keys 方法获取字典中所有的键；使用 values 方法获取字典中所有的值， update 方法实现两个字典的合并操作；跟列表一样，从字典中删除元素也可以使用 del 关键字。
