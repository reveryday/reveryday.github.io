Tally Card

Simple Data-card Form: Fn:P s1 : : : sK
or
General Data-card Form: Fn:P s1 (s2 : : : s3) (s4 : : : s5) s6 s7 : : : [T]

n : Tally number. Restriction: n 99999999
P : Particle designator ( 1 ).
sk : Problem number of surface or cell for tallying ( 2 ).
T : Total over specified surfaces for F1 tallies; average over specified surfaces or cells for F2 , F4 , F6 , and F7 tallies. (Optional) ( 3 )

计数卡 类型 输出物理量 单位
F1 表面电流 粒子电流 粒子/源粒子
F2 表面通量 表面平均通量 粒子/cm²/源粒子
F4 单元通量 单元平均通量 粒子/cm²/源粒子
F5 点探测器 点通量 粒子/cm²/源粒子
F6 能量沉积 单位质量能量沉积 MeV/g/源粒子
F7 裂变能量沉积 裂变能量 MeV/源粒子
F8 脉冲高度 能量沉积分布 MeV 或概率
