Tally Card

Simple Data-card Form: Fn:P s1 : : : sK
or
General Data-card Form: Fn:P s1 (s2 : : : s3) (s4 : : : s5) s6 s7 : : : [T]

n : Tally number. Restriction: n 99999999
P : Particle designator ( 1 ).
sk : Problem number of surface or cell for tallying ( 2 ).
T : Total over specified surfaces for F1 tallies; average over specified surfaces or
