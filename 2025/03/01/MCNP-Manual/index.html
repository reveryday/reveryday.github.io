Input Card

Tally Card

Simple Data-card Form: Fn:P s1 : : : sK
or
General Data-card Form: Fn:P s1 (s2 : : : s3) (s4 : : : s5) s6 s7 : : : [T]
n Tally number. Restriction: n 99999999
P Particle designator
sk Problem number of surface or cell for tallying
T Total over specified surfaces for F1 tallies; average over specified surfaces or cells for F2 , F4 , F6 , and F7 tallies. (Optional)

WWE

WWG: Weight-window Generation

WWG 802 1 0 J J J J 0 - 这是一个权重窗口生成卡，用于 802 号记数
wwe:p 1.0000E+02 - 光子权重窗口的能量上限，设置为 100
wwn1:p 部分包含了权重窗口的下限值，这些值随能量变化，结尾的 -1.0000E+00 是数据块的终止符
